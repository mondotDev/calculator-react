# calculator-react
## Set Up
 - ```npx create-react-app calculator-react```
 - ```cd calculator-react```
 - ```npm start```
 - Clean up public and src folders:

        | /public       | /src      |
        |---------------|-----------|
        | favicon.ico   | App.js    |
        | index.html    | index.css |
        | manifest.json | index.js  |
        | robots.txt    |           |
